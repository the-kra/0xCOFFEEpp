# ☕ 0xC0FFEE++

> Die digitale Kaffee-Stempelkarte im Holzkisten-Design

Eine elegante iOS-App, die das alte Konzept der Kaffee-Stempelkarte neu erfindet — mit einer animierten Holzkiste, Goldmünzen und echtem Kassenklingeln.

🌐 **Website:** [theodorkranz.github.io/0xC0FFEEpp](https://theodorkranz.github.io/0xC0FFEEpp/)
🔒 **Datenschutz:** [Privacy Policy](https://theodorkranz.github.io/0xC0FFEEpp/privacy.html)
📋 **Support:** [FAQ & Kontakt](https://theodorkranz.github.io/0xC0FFEEpp/support.html)

---

## 🎯 Was die App macht

- **Tippe** auf die Holzkiste → sie öffnet sich animiert
- **Tippe** auf den goldenen Coin → +1 Kaffee zählt, mit Münz-Sound und Vibration
- **Lange drücken** auf den Coin → Bezahl-Dialog mit Kassen-Klingeln
- **Lange drücken** auf den Zähler → Preis pro Kaffee einstellen

Alles läuft **komplett offline**, ohne Server, ohne Tracking, ohne Werbung.

---

## ✨ Features

- 🪵 **Animierte Holzkiste** mit Schiebetüren, Maserung und Ast-Details
- 🪙 **Goldmünze** mit Logo und Glanz-Animationen
- 📳 **CoreHaptics** für satte Vibrationen
- 🔊 **Echte Münz- & Kassen-Sounds**
- ⚙️ **Anpassbarer Preis** (0,10 € bis 99,99 €)
- 🔒 **100% privat** — alle Daten bleiben lokal
- 📱 **Querformat-Design** für den Tresen-Einsatz

---

## 🛠 Tech Stack

- **SwiftUI** (iOS 17+)
- **AVFoundation** für Sound-Wiedergabe
- **CoreHaptics** für haptisches Feedback
- **@AppStorage** für lokale Datenspeicherung

## 🖨 3D-Druck-Halterung
